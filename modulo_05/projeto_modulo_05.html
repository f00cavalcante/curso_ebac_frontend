Projeto 01: Game Shop

Neste módulo, foi desenvolvido um projeto prático referente a criação de uma página web de "Game Shop". O intuito deste módulo seria apenas trazer a aplicabilidade dos conhecimentos de HTML e CSS adquiridos até aqui.
Desta forma, esta página será apenas para realizar esta descrição, e o acesso ao projeto desenvolvido.

Para acessar o resultado do projeto, clique neste link.

Exercício

No exercício proposto, seria a criação de uma nova página, seguindo aquilo que foi desenvolvido neste módulo 05, com tema a livre escolha. Desta forma, optei por efetuar a criação de uma página contendo um formulário, que após preenchimento, trará o retorno da interação realizadas através de algumas interações por meio do script JavaScript criado. Desta forma, foi aplicado alguns quesitos, estabelecidos, como:

  • Conter um formulário;
  • Conter imagem;
  • Conter customização para títulos e parágrafos.

O resultado poderá ser consultado neste link.
