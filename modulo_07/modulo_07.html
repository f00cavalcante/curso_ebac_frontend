Integração JavaScript com HTML

Neste módulo está sendo desenvolvido o processo de integração do JavaScript com o HTML. Por meio desta linguagem de programação é possível efetuar a captura de todos os elementos no documento web, bem como seus atributos.

Capturar as informações dos elementos torna-se útil para gerar validações nos sistemas web a serem construídos, ou mesmo inserir valores.

No módulo 07, em questão, foi realizado durante as aulas, a construção de um formulário simples com validações que visam gerar a simulação de depósitos bancários.
